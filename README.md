# Хлызов Ефим БАС-1 Лабораторная №8

## Задание 1. 

### Текст задачи 

Разработать консольное приложение с дружественным интерфейсом с возможностью выбора
заданий для работы с «базой данных (БД)», хранящейся в бинарном файле. Перечень полей
(минимум 5), достаточно полно характеризующих заданную в варианте предметную область,
предложить самостоятельно (постараться отразить в перечне полей такие, которые требуют разных
типов данных). Приложение должно выполнять следующие функции:
1. Чтение базы данных из бинарного файла.
2. Просмотр базы данных.
3. Удаление элементов (по ключу).
4. Добавление элементов.
5. Реализация 4 запросов (формулировки запросов придумать самостоятельно). 2 запроса
должны возвращать перечень, 2 запроса одно значение.
В классе должны присутствовать свойства, конструкторы, перегруженный метод ToString(). Весь
функционал приложения реализовать в виде методов вспомогательного класса с помощью LINQ-
запросов.
Предусмотреть обработку возможных ошибок при работе программы.
Пункт 5 оценивается в 2 балла, остальные пункты по 1 баллу. Максимально за лабораторную работу
можно получить 6 баллов.


### Алгоритм решения

Программа представляет собой консольное приложение для работы с базой данных магазина продуктов, хранящейся в бинарном файле. После запуска выполняется загрузка данных из файла в список объектов класса Product. Пользователю предоставляется меню для просмотра базы данных, добавления новых товаров, удаления товаров по идентификатору и выполнения запросов. Перед созданием объекта производится проверка корректности вводимых данных с помощью методов класса InputHelper. Для хранения информации используется бинарный файл, запись и чтение которого выполняются средствами BinaryWriter и BinaryReader. Все операции поиска и обработки данных реализованы с использованием LINQ-запросов. Перед завершением работы программы база данных сохраняется в файл. Для предотвращения аварийного завершения приложения предусмотрена обработка исключений при работе с файлами.
